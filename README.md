# HR-Analytics-PowerBI-Dashboard
HR ANALYTICS: HR analysis insights dashboard.
The HR Analytics Dashboard project provides data-driven insights into employee attrition, workforce demographics, job satisfaction, and performance metrics. Using data visualization and analytical techniques, the dashboard helps identify patterns affecting employee retention and organizational productivity. It enables HR teams to monitor key metrics such as attrition rates, department-wise employee distribution, age groups, salary trends, and job roles to support informed decision-making. This project demonstrates how business intelligence tools can transform raw HR data into actionable insights for improving employee engagement and workforce management.
TECH STACK:
Power BI – For creating interactive dashboards and data visualizations.
Microsoft Excel (CSV/XLSX) – Used as the data source for storing HR-related datasets.
Power Query – For data cleaning, transformation, and preprocessing.
DAX (Data Analysis Expressions) – Used to create calculated columns, measures, and KPIs such as Attrition Rate and Average Salary.
Data Modeling – To establish relationships between tables and optimize data analysis.
Power BI Service (Optional) – For publishing and sharing reports online.
DATA SOURCE:
The data used in this HR Analytics project is sourced from a structured HR employee dataset in Microsoft Excel (XLSX/CSV) format. The dataset contains comprehensive employee information, including age, gender, department, job role, education level, monthly income, years at the company, job satisfaction, work-life balance, and attrition status. These attributes are used to analyze workforce trends, identify factors contributing to employee attrition, and generate actionable insights through interactive Power BI dashboards. The dataset enables the visualization of key HR metrics such as attrition rates, employee demographics, salary distribution, and departmental performance, supporting data-driven decision-making in human resource management.
Features/Highlights:
->Developed an interactive HR Analytics Dashboard to monitor key workforce metrics and employee trends.
->Performed employee attrition analysis to identify departments, job roles, and demographics with higher turnover rates.
->Visualized employee distribution based on age, gender, education, and department for better workforce insights.
->Analyzed salary patterns, job satisfaction levels, and work-life balance factors influencing employee retention.
